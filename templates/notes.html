Some notes on the readme file

This is a workout tracker, based around high-intensity workouts, specifically the Tabata protocol.
HIIT provides both aerobic and anaerobic conditioning, meaning it improves your cardio and your strength in one workout.

The workout dropdown above contains lists of exercises, muscle and equipment that can be added to via forms.

The history page provides a view of all exercises completed and their respective metrics.
The routine page allows a user to build a list of exercises and track the reps, sets and weights used.

Future Stuff

The data stored will eventually be used to understand potential imbalances for anterior/posterior chains and muscle groups.
It will also include a working calendar (concept below) and an inhouse timer to help with managing workouts.
